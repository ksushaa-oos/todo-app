# To-Do App (Java)

Консольное приложение для управления списком задач.

## Функционал:
- Добавление задач
- Просмотр списка задач
- Удаление задач

## Технологии:
- Java
- ООП
- Коллекции (ArrayList)

## Описание
Проект был создан для практики Java и закрепления знаний по работе с коллекциями и пользовательским вводом.
